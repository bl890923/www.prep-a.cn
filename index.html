上海阶点国际学校

格物致知、睿思进取

上海阶点国际学校是一家专业从事中美国际教育的高中。理念是“格物致知、睿思进取”，方针是“中学为体、西学为用”，致力于培养具有中国底蕴的世界公民。目前与国外多所高中与大学建立了深度合作，在美国纽约、阿肯色州设立教育中心，提供一站式留学服务。

办学优势

美国Grace Academy授权学校，美式GPA学分修读标准，中美学分互认。

教学优势

国家课程国际化、国际课程校本化、学生辅导个性化、评价多元化。

师资优势

外教来自英语母语国家，中文教师具有留学背景与丰富经验。

学部介绍

高中部

阶点国际学校高中阶段，以国家课程标准为主体，引入美国格瑞斯学校优质课程，培养“具有中国底蕴的世界公民”。

招生对象：年满十六周岁
入学时间：9月
录取方式：笔试+面试

课程设置

国标课程

语文、数学、英语 (语文/数学中文授课)

国际课程

美方英语、音乐、美术、体育，全英文授课

创新课程

主题式研究性学习课，双语授课

素质课程

体育、艺术、科技、实践体验活动课

课程优势

  • ✓ 美国合作高中认证毕业证书
  • ✓ 美方定期监督教学质量
  • ✓ 在校攻克托福、SSAT
  • ✓ 重视国学及艺术融合
  • ✓ 全日制小班教学
  • ✓ 丰富社会实践与游学

中美高中大学2+2+2连读国际班

2+2+2项目由阶点国际学校与美国格瑞斯学校共同承担，学生高二毕业后赴美读大一、大二，同时获美国高中毕业证及副学士学位，之后转升顶尖名校大三，省时省钱。

100%赴美留学 · 可升入美国前50名校

招生对象：初中毕业或高一在读，有出国意向，全国招生不限户籍。录取：笔试+面试（英语、数学、理化）。

✓ 有准备: 3年制连读课程
✓ 保录取: 美国优秀大学双录取
✓ 省时间: 4年获学士学位，节省1-2年
✓ 免托福/SAT: 凭在校ESL/GPA成绩申请
✓ 费用低: 在美总费用约80-100万，省30万+
